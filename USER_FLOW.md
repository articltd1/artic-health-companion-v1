# User Flow

## Goal
Define the ideal product experience flow for patients, doctors, and administrators in ARTIC Health Companion.

## Patient Flow
1. Landing page
   - Hero overview
   - Key benefits: AI health prediction, care coordination, trusted doctors
   - Primary CTA: Get started / Sign in
2. Authentication
   - Login / Register
   - Social proof and trust signals
3. Patient Dashboard
   - Health summary
   - Upcoming appointments
   - AI recommendations
   - Quick actions: Book doctor, predict condition, view record
4. Disease Prediction
   - Symptom entry
   - Demographic input
   - Prediction insights
   - Recommended next steps
5. Doctor Booking
   - Search/filter doctors by specialty and availability
   - Doctor profile detail
   - Book appointment flow
6. Appointments
   - Appointment status tabs
   - Active appointment details
   - Reschedule or cancel actions
7. Profile & Records
   - Personal information
   - Medical record summary
   - Care team and history
   - Settings

## Doctor Flow
1. Doctor login
   - Reassuring, secure access
2. Doctor dashboard
   - Today's schedule
   - Patient risk summary
   - Unresolved cases
   - Quick actions: review patient record, confirm appointment
3. Appointment work queue
   - Filter by status and priority
   - Patient details preview
   - Action buttons (accept, review, message)
4. Patient record access
   - Medical history overview
   - Visit summary
   - Relevant AI insights
5. Profile management
   - Availability
   - Contact and specialization
   - Account settings

## Admin / Hospital Flow
1. Admin login
   - Secure sign-in and institution selection
2. Admin dashboard
   - System health metrics
   - Patient volume and utilization
   - Doctor coverage and appointments
   - AI performance metrics
3. Patient summary
   - Active patient cohorts
   - Alerts and compliance flags
4. Provider management
   - Doctor roster
   - Appointment capacity
   - Team settings
5. Reports & analytics
   - Usage trends
   - Outcomes and engagement data

## Cross-Functional Flow
- Trust signals across flows
  - Security badge
  - Data privacy note
  - Clinical oversight
- AI explanation flow
  - What the model used
  - Confidence levels
  - Recommended next steps
- Accessibility flow
  - Keyboard and screen reader navigation
  - Clear labels and error handling
